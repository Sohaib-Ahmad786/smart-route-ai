# 🚀 Smart Route AI

A modern **Traffic Intelligence & Route Optimization Dashboard** built to simulate real-time route analytics with an interactive and responsive user interface.

---

## 📌 Overview

Smart Route AI is designed to provide **intelligent route suggestions** by analyzing simulated traffic conditions, weather patterns, and safety metrics.
The application focuses on delivering a **real-world dashboard experience** using modern frontend technologies.

---

## ✨ Key Features

* 📊 **Dynamic Traffic Visualization**

  * Interactive charts powered by ApexCharts
  * Smooth animations with real-time-like data simulation

* 🚦 **Route Intelligence System**

  * Traffic congestion analysis
  * Weather condition insights
  * Safety score estimation

* 🖱️ **Interactive User Experience**

  * Fully functional form handling
  * Loading states & smooth transitions
  * Responsive feedback system

* 🎨 **Modern UI/UX**

  * Clean dark-themed dashboard
  * Glassmorphism-inspired components
  * Fully responsive across devices

---

## 🛠 Tech Stack

| Category   | Technology      |
| ---------- | --------------- |
| Frontend   | Next.js (React) |
| Styling    | Tailwind CSS    |
| Animations | Framer Motion   |
| Charts     | ApexCharts      |

---

## 🧩 System Architecture

* **RouteForm** → Handles user input
* **RouteLogic** → Processes route calculations
* **RouteResult** → Displays analytics
* **RouteChart** → Visualizes traffic data
