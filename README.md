# Smart Route AI 🚀

A modern Smart Traffic & Route Suggestion Web App built for internship.

## 🔥 Features
- Smart route suggestion
- Animated route visualization 🚗
- Clean modern UI (Dark Theme)
- Fully responsive design 📱
- Smooth animations using Framer Motion

## 🛠 Tech Stack
- Next.js (React)
- Tailwind CSS
- Framer Motion

## 📦 Modules
- Route Form (User Input)
- Route Result (Summary)
- Route Visualization (Animated Path)
